NgRx Selectors

Definition

Selectors are pure functions used for obtaining slices of store state. They provide a way to query your store for data.

Selectors are composable, reusable, and memoized for performance. They help keep components decoupled from the store structure.

Use selectors to compute derived data, allowing Redux to store the minimal possible state.

Selector Implementations
Basic Selectors
// Feature selector
export const selectItemState = 
  createFeatureSelector<ItemState>('items');

// Basic selector
export const selectAllItems = createSelector(
  selectItemState,
  (state: ItemState) => state.items
);

// Computed selector
export const selectItemsCount = createSelector(
  selectAllItems,
  (items) => items.length
);
Advanced Selectors
// Parameterized selector
export const selectItemById = (id: number) => 
  createSelector(
    selectAllItems,
    (items) => items.find(item => item.id === id)
  );

// Composed selector
export const selectLatestItem = createSelector(
  selectAllItems,
  (items) => items.length > 0 
    ? items[items.length - 1] 
    : null
);
