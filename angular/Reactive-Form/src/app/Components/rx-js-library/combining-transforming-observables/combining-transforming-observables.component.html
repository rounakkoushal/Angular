Combining & Transforming Observables

Definition

RxJS provides powerful operators to combine multiple Observables and transform their emissions in various ways.

Combining operators like merge, concat, zip, and combineLatest allow you to work with multiple data streams simultaneously.

Transformation operators like map, scan, reduce, and groupBy help reshape and process Observable emissions to meet your application needs.

Combining Operators
  • merge - Concurrent combination
  • concat - Sequential combination
  • zip - Pair-wise combination
  • combineLatest - Latest values combination
Transformation Operators
  • map - Transform values
  • scan - Accumulate over time
  • reduce - Single accumulated result
  • groupBy - Group by key

Code Examples

Implementation Examples
Merge Example
// Merge multiple streams
const stream1$ = interval(1000).pipe(map(i => `A${i}`));
const stream2$ = interval(1500).pipe(map(i => `B${i}`));

merge(stream1$, stream2$).subscribe(console.log);
// Output: A0, B0, A1, A2, B1, A3...
CombineLatest Example
// Combine latest values
const name$ = of('John', 'Jane');
const age$ = of(25, 30);

combineLatest([name$, age$]).subscribe(([name, age]) => {
  console.log(`${name} is ${age} years old`);
});
Zip Example
// Zip streams together
const numbers$ = of(1, 2, 3);
const letters$ = of('a', 'b', 'c');

zip(numbers$, letters$).subscribe(([num, letter]) => {
  console.log(`${num}${letter}`);
});
// Output: 1a, 2b, 3c
Scan Example
// Accumulate values over time
of(1, 2, 3, 4, 5).pipe(
  scan((acc, curr) => acc + curr, 0)
).subscribe(console.log);
// Output: 1, 3, 6, 10, 15
Operator Comparison
merge

Concurrent emission from multiple sources

concat

Sequential emission, one after another

zip

Pair values from sources in order

combineLatest

Latest value from each source
