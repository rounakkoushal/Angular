Error Handling in RxJS

Definition

Error handling in RxJS allows you to gracefully manage errors that occur in observable streams without breaking the entire application flow.

RxJS provides several operators to catch, handle, and recover from errors, ensuring your application remains stable and user-friendly.

Proper error handling prevents unhandled exceptions and provides fallback mechanisms for failed operations.

Error Handling Operators
catchError Catch and handle errors gracefully
retry Retry failed operations automatically
retryWhen Retry with custom logic and delays
throwError Create error observables for testing
Common Error Scenarios
  • HTTP request failures
  • Network connectivity issues
  • Invalid data transformations
  • Authentication failures
  • Timeout errors

Error Handling Implementations
CatchError Example
// Basic error handling
this.http.get('/api/data').pipe(
  catchError(error => {{ '{' }}
    console.error('API Error:', error);
    return of([]); // Return fallback value
  })
).subscribe(data => {{ '{' }}
  console.log('Data:', data);
});

// Error handling with different responses
source$.pipe(
  catchError(err => {{ '{' }}
    if (err.status === 404) {{ '{' }}
      return of('Not found');
    }
    return throwError(err);
  })
);
Retry Examples
// Simple retry
this.http.get('/api/data').pipe(
  retry(3), // Retry up to 3 times
  catchError(error => {{ '{' }}
    return of('Failed after retries');
  })
).subscribe();

// Retry with delay
source$.pipe(
  retryWhen(errors => 
    errors.pipe(
      delay(1000), // Wait 1 second
      take(3) // Max 3 retries
    )
  )
);
Advanced Error Handling
// Multiple error strategies
source$.pipe(
  retry(2),
  timeout(5000),
  catchError(error => {{ '{' }}
    if (error.name === 'TimeoutError') {{ '{' }}
      return of('Request timed out');
    }
    // Log error and return fallback
    this.logger.error(error);
    return of(this.getFallbackData());
  })
).subscribe();
Error Recovery
// Graceful degradation
this.getData().pipe(
  catchError(() => this.getCachedData()),
  catchError(() => this.getDefaultData()),
  catchError(() => {{ '{' }}
    this.showErrorMessage();
    return EMPTY;
  })
).subscribe(data => {{ '{' }}
  this.displayData(data);
});
