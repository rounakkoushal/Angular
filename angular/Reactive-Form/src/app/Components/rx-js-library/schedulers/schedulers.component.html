RxJS Schedulers

Definition

Schedulers in RxJS control when and how observables emit values and when subscriptions are executed.

They provide a way to control the execution context and timing of observable operations, enabling you to optimize performance and manage concurrency.

RxJS provides several built-in schedulers for different use cases: synchronous execution, asynchronous execution, animation frames, and more.

Built-in Schedulers
  • asyncScheduler - Asynchronous execution
  • queueScheduler - Synchronous queue
  • animationFrameScheduler - Browser animation frames
  • asapScheduler - As soon as possible
Common Use Cases
  • Controlling execution timing
  • Managing concurrency
  • Optimizing performance
  • Testing with virtual time
  • Animation synchronization

Code Examples

Implementation Examples
AsyncScheduler Example
// Using asyncScheduler
import { asyncScheduler } from 'rxjs';
import { observeOn } from 'rxjs/operators';

of(1, 2, 3).pipe(
  observeOn(asyncScheduler)
).subscribe(console.log);

// Schedule a task
asyncScheduler.schedule(() => {
  console.log('Async task executed');
}, 1000);
QueueScheduler Example
// Using queueScheduler
import { queueScheduler } from 'rxjs';

of(1, 2, 3).pipe(
  observeOn(queueScheduler)
).subscribe(console.log);

// Synchronous execution
queueScheduler.schedule(() => {
  console.log('Queue task executed');
});
AnimationFrameScheduler
// For smooth animations
import { animationFrameScheduler } from 'rxjs';

interval(0, animationFrameScheduler).pipe(
  map(i => i * 2),
  take(60)
).subscribe(value => {
  // Update animation
  element.style.left = value + 'px';
});
AsapScheduler Example
// Execute as soon as possible
import { asapScheduler } from 'rxjs';

of(1, 2, 3).pipe(
  observeOn(asapScheduler)
).subscribe(console.log);

// High priority execution
asapScheduler.schedule(() => {
  console.log('ASAP task executed');
});
Scheduler Comparison
asyncScheduler

Uses setTimeout for asynchronous execution with delays

queueScheduler

Synchronous execution in queue order, no delays

animationFrame

Syncs with browser's repaint cycle for smooth animations

asapScheduler

Uses Promise.resolve() for microtask execution
